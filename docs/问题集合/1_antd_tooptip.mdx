:::danger
问题描述：tooptip/poprove 等提示，在异步加载时，`加载中···`的过度内容窗口会在`加载完毕显示异步内容`时重新计算窗口大小、位置，会有一定概率加载不出来

> 实际效果:

> > 1.先显示`加载中···`

> > 2.获取异步数据后,窗口直接消失

> > 3.一般出现在 tooltip 靠近页面最下方时出现问题

:::

<details>
  <summary>初步解决方法</summary>
  <div>法1 自己重写ToolTip组件；</div>
  <hr />
  <div>
    法2 自己预估好异步加载之后的浮窗宽度或高度，把{" "}
    <span style={{ color: "red" }}>加载中...</span>{" "}
    的过度窗口的高度或宽度预设成异步加载之后的大小，方便浮窗定位
  </div>
</details>

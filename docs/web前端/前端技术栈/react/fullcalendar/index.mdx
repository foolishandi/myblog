## 1.官网与简介

[fullcalendar.io/react](https://fullcalendar.io/docs/react)

[fullcalendar.io/docs](https://fullcalendar.io/docs)

几乎可以包含日历、日程、跨天计划等所有的需求，但是每日事项超出`max` 的 `Event popover` 缺少相关 `api`，这个浮窗事件不好控制，可以借助`MutationObserver`监听浮窗出现进行相关操作

## 2.安装

```
    npm install @fullcalendar/react @fullcalendar/core @fullcalendar/daygrid
```

    const callback = function (mutationsList: any, observer: any) {

            addEle.style.top = "0";
            addEle.style.left = "0";
            parentEle.style.position = "relative";
            parentEle?.appendChild(addEle);

        }
      }

    const observer = new MutationObserver(callback);
